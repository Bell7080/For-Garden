# For-Garden

A/D 키로 좌우 이동하는 스프라이트 러너 데모입니다.

## 실행
정적 파일이므로 아래처럼 실행하면 됩니다.

```bash
python -m http.server 8080
```

브라우저에서 `http://localhost:8080` 접속.

## GitHub Pages (main 브랜치 /root 배포)
- `main` 브랜치의 **/root(저장소 루트)** 기준으로 배포할 때 그대로 동작합니다.
- 루트에 `index.html`, `styles.css`, `game.js` 파일을 유지하세요.
- 스프라이트 파일 `char_run_001.png`는 바이너리 PR 이슈 때문에 저장소에서 제외했으니, 배포 전에 직접 루트에 업로드하세요.

## 스프라이트 시트 규격
- 파일명: `char_run_001.png`
- 시트 분할: 8열 x 2행 (총 16칸)
- 사용 프레임: 1~15번만 루프
- 미사용 프레임: 16번

즉, 재생 순서는 **1 → 2 → ... → 15 → 1** 입니다.
